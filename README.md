# embodied-metal-ops

Shareable assets and ops for **The Embodied Metal Hackathon** (July 17–19, Mission Robotics, SF) and Mission Robotics events.

This repo is public so assets are linkable — hand a raw URL to a judge, a sponsor, or drop one into a Luma cover.

## What's here

### `assets/social/`
Announcement and event cards, rendered at OG (1200×630) and square (1080×1080):

| File | Use |
|---|---|
| `hackathon-announce-{og,square}.png` | The hackathon announcement card |
| `judge-chase-{og,square}.png` · `judge-junfan-{og,square}.png` | Per-judge "I'm judging" cards |
| `sponsor-rerun-{og,square}.png` | Rerun sponsor card *(hold until announce)* |
| `hacker-accepted-{og,square}.png` | "I'm in" card for accepted hackers to post |
| `event-openhouse-{og,square}.png` | Residency × Mission Robotics Open House (Jun 29) |

## Grabbing a raw URL

Any file is hostable via:

```
https://raw.githubusercontent.com/mission-robotics-ai/embodied-metal-ops/main/assets/social/<file>.png
```

Use these for Luma covers, embeds, or anywhere you need a hosted image link.
